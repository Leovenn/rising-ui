## 使用包管理器

建议您使用包管理器 (NPM, Yarn, pnpm) 安装 RisingUI, 然后您就可以使用打包工具，例如 Vite 和 webpack

```
# 选择一个你喜欢的包管理器

# NPM
$ npm install rising-ui --save

# Yarn
$ yarn add rising-ui

# pnpm
$ pnpm install rising-ui
```
